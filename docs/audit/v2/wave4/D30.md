### Critical Path

The single **C-severity** finding (reel-remix param mismatch, #28) is duplicated as an H-severity in reel/[id].tsx (#17) because the bug originates from the caller. The remix feature is 100% broken: the reel detail screen sends `reelId` but reel-remix reads `originalReelId`.

### Systemic Issues

1. **Light mode broken everywhere**: All 5 screens use hardcoded `colors.text.*` (53 total instances) instead of theme-aware `tc.text.*`. Every screen is illegible in light mode.
2. **No keyboard handling on form screens**: reel-remix and report have TextInputs in ScrollViews without KeyboardAvoidingView.
3. **Module-level Dimensions**: 4 of 5 screens compute layout constants from `Dimensions.get('window')` at module scope. None listen for dimension changes.
4. **fontWeight instead of fontFamily**: All screens except ramadan-mode use raw `fontWeight` strings instead of the project's `fonts.*` system.
5. **Missing haptics on report screen**: Zero haptic feedback on any interaction.
