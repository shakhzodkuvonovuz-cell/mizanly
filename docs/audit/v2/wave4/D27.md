### Top Themes

1. **Light mode broken across ALL 5 screens** (H): Every screen uses `colors.text.*` hardcoded dark-mode values in stylesheets despite receiving `tc` from `useThemeColors()`. This is a systematic bug.

2. **RTL broken on 4 of 5 screens** (H/M): `prayer-times.tsx` and `post-insights.tsx` don't even destructure `isRTL`. `playlists/[channelId].tsx` destructures but never uses it. Only `product/[id].tsx` partially handles RTL but has hardcoded strings.

3. **Hardcoded English strings in `product/[id].tsx`** (H): 6 user-visible strings not wrapped in `t()` — "Halal Certified", "Muslim-owned", "Description", "Seller", "Payment Options", "Reviews".

4. **Silent error handling** (C/M): `product/[id].tsx` order mutation has no `onError` (data loss risk). `post-insights.tsx` swallows API errors showing blank screen.

5. **No haptic feedback** on playlists and post-insights screens (M).

6. **No `staleTime`** on any query across all 5 screens — every focus triggers unnecessary refetches.
