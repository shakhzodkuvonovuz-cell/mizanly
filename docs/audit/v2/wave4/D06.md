### Critical patterns across all 5 screens:
1. **Light theme completely broken** — Every screen has 10-60+ hardcoded `colors.dark.*` and `colors.text.*` values in StyleSheet. JSX overrides some with `tc.*` but stylesheets are all dark-only.
2. **RTL support missing or incomplete** — `caption-editor`, `cashout`, `channel/[handle]`, and `charity-campaign` have little to no RTL consideration. `challenges.tsx` is the only one with proper `rtlFlexRow`/`rtlTextAlign`.
3. **No offline handling** — All 5 screens lack offline detection or messaging.
4. **Silent error swallowing** — Mutations and queries frequently lack error handlers (toast/retry).
5. **`cashout.tsx` is entirely non-functional** — Backend endpoints don't exist, screen has no feature gate.
