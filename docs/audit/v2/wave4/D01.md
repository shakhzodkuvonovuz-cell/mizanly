### Critical Patterns Across All 5 Screens

1. **Light mode broken everywhere:** All 5 screens use `colors.text.*` / `colors.dark.*` in static StyleSheet instead of `tc.*` from `useThemeColors()`. Light mode will show dark-theme colors on a light background.

2. **Missing haptic feedback:** 2fa-setup and 2fa-verify have zero haptic usage. Account-switcher is missing haptics on key actions.

3. **No KeyboardAvoidingView on input screens:** Both 2fa-setup and 2fa-verify have TextInput fields but no keyboard avoidance. On iOS, the keyboard will cover the inputs.

4. **SafeAreaView missing on 3/5 screens:** 2fa-setup imports but doesn't use it. 2fa-verify and achievements don't have it at all. All three use hardcoded `paddingTop: 100`.

5. **Dead/stub features:** Account-switcher has 3 non-functional controls (manage accounts, default account, auto-switch toggle). 2fa-verify has a dead "Contact Support" button. 2fa-setup's authenticator app picker is decorative.
