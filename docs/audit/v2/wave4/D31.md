### Critical Patterns Across All 5 Screens

1. **Light mode broken (A1):** 4 of 5 screens use `colors.text.*` or `colors.dark.*` in StyleSheet instead of `tc.text.*`. Only `safety-center.tsx` gets this right.
2. **Missing haptic feedback (B10):** 4 of 5 screens lack `useContextualHaptic()`. Only `safety-center.tsx` uses it.
3. **Silent error swallowing (C23):** `revenue.tsx` catches ALL fetch errors silently — user sees stale/empty data with zero indication of failure. This violates the "All errors explicit, never silently swallowed" rule.
4. **No staleTime (C27):** 3 screens using React Query have no `staleTime`, causing unnecessary refetches on every focus.
5. **Revenue uses manual state instead of React Query (C36):** `revenue.tsx` implements its own fetch/cache/pagination with `useState`+`useCallback` instead of `useQuery`/`useInfiniteQuery`, losing all React Query benefits.
6. **Trend icon bug (F45):** `revenue.tsx` line 152 renders `trending-up` for both up AND down trends.
