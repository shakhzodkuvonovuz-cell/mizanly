### Critical Patterns Across All 5 Screens

1. **Light mode broken (all 5 screens):** Every screen uses `colors.text.*` in StyleSheet instead of `tc.text.*`, making all text invisible in light theme. This is the single most impactful pattern -- affects ~40 style properties total.

2. **No double-tap guards (4/5 screens):** starred-messages, sticker-browser, stitch-create, and sound all lack double-tap protection on mutation/navigation actions.

3. **`isRTL` unused (3/5 screens):** sound, starred-messages, and sticker-browser destructure `isRTL` but never use it. `borderLeft` instead of `borderStart` in starred-messages is an actual RTL bug.

4. **Stale `Dimensions.get` at module scope (2/5 screens):** sound and stitch-create compute sizes at import time, breaking on resize/rotation.
