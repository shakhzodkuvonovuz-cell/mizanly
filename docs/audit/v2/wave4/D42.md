### Top 3 Must-Fix

1. **#40 / #1-3 / #30-31: Light theme is broken** across risalah.tsx and _layout.tsx. StyleSheet colors are dark-mode-only. Tab bar blur and Android bg are hardcoded dark. Every text color in ConversationRow (`chatName`, `chatTime`, `chatPreview`, `filterChipText`) uses `colors.text.*` directly instead of `tc.text.*`.

2. **#16: `isPinned` always false** because `Conversation` type lacks the field. The code uses an unsafe `as unknown as Record<string, unknown>` cast that silently returns `undefined`. Pin/unpin toggle is non-functional.

3. **#11-12 / #17-18: Swipe actions have no error handling or user feedback.** Archive mutation has no `onError`. Pin is fire-and-forget. Neither provides a toast. Pin also has a race condition (invalidates cache before API responds).
