### Critical finding (C):
- **#45 (send-tip.tsx):** Double-tap on "Send Tip" button can create multiple PaymentIntents, charging real money multiple times. Needs a `useRef`-based lock or mutation-based `isPending` guard.

### Systemic issue — A1-Light:
Nearly every screen uses `colors.text.primary` (#FFF) and `colors.text.secondary` (#9BA4AE) directly in StyleSheet instead of `tc.text.primary` / `tc.text.secondary` from `useThemeColors()`. This makes **all 5 screens functionally broken in light mode** — text is invisible white-on-white or near-invisible gray-on-white. This is a bulk fix: replace all `colors.text.*` references in StyleSheet.create with the equivalent `tc.text.*` from the theme-aware hook.
