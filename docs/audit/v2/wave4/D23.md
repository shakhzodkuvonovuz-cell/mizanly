### Critical patterns across all 5 screens:
1. **Static `colors.dark.*` / `colors.text.*` in StyleSheet** (28 instances) — every screen uses hardcoded dark-theme colors in static stylesheets. Many are overridden at runtime with `tc.*` in JSX, but the static values remain as dead fallbacks or, worse, are NOT overridden and will break in light mode.
2. **No RTL support** on 4/5 screens (all except `media-settings.tsx`) — Arabic, Urdu users see broken layouts.
3. **No search debounce** on marketplace and mentorship — every keystroke fires API requests.
4. **No KeyboardAvoidingView** on 3 screens with TextInputs (marketplace, membership-tiers, mentorship).
5. **Mentorship content behind GlassHeader** — no safe area or top padding to account for absolute-positioned header.
