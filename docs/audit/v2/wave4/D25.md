## Key Themes

1. **Light mode is catastrophically broken across all 5 screens.** Every screen uses `colors.text.primary/secondary/tertiary` (dark-mode hardcoded white/gray values) in stylesheets instead of `tc.text.*` from `useThemeColors()`. This makes virtually all text invisible on white backgrounds. 18 H-severity findings are for this single pattern.

2. **RTL support is missing from 3 of 5 screens.** `nasheed-mode.tsx`, `new-conversation.tsx`, and `notification-tones.tsx` have zero RTL handling. For an app targeting Arabic, Urdu, and Malay speakers, this is a significant gap.

3. **No haptic feedback on 2 screens.** `nasheed-mode.tsx` and `new-conversation.tsx` don't import or use `useContextualHaptic` at all.

4. **Double-tap protection is inconsistent.** Multiple screens allow rapid repeated taps to fire mutations multiple times. `notifications.tsx` has a particularly bad case where the mark-all-read button fires twice per single tap due to dual `onPress` handlers.
