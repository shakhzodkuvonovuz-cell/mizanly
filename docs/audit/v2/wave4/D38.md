### Top Issues by Theme

1. **Light mode broken (A1)**: 5 screens all use `colors.text.*` directly instead of `tc.text.*`. Every screen will have invisible or wrong-colored text in light mode. This is the single most pervasive issue.

2. **Double-tap / concurrent mutations (B13/C30)**: Video export, premiere scheduling, voice post publish, and volunteer sign-up all lack robust double-tap guards. React-query's `isPending` is not synchronous enough to prevent rapid-fire taps.

3. **Silent error swallowing**: Voice post create has a completely empty catch on recording start. Voice recorder has empty catch in the level polling interval. Volunteer board has no `onError` on sign-up mutation.

4. **SafeArea inconsistency (A3)**: Video editor omits bottom safe area. Voice post create has no SafeAreaView at all. Voice recorder double-counts top inset. Volunteer board has no SafeAreaView.

5. **Keyboard handling (B19)**: Video premiere and video editor have TextInputs inside ScrollViews with no keyboard avoidance.
