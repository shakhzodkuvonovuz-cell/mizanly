### Critical Patterns Across All 5 Screens

1. **Light theme broken everywhere**: All 5 screens use `colors.dark.*` or `colors.text.*` in static stylesheets instead of `tc.*`. The pattern of inline `tc` overrides covers containers but misses text colors, borders, and backgrounds in sub-components.

2. **RTL ignored**: 4 of 5 screens have no RTL handling. `hashtag/[tag].tsx` destructures `isRTL` but never uses it. `hifz-tracker.tsx` is the only one that actually uses `rtlFlexRow`/`rtlTextAlign`.

3. **No SafeAreaView on 2 screens**: `hashtag/[tag].tsx` and `image-editor.tsx` use hardcoded padding instead of safe area insets.

4. **image-editor is non-functional**: The sliders cannot be dragged (no gesture handler), and "Done" discards all edits. This is a UI mockup, not a working editor.

5. **Missing error states**: `hifz-tracker.tsx` (3 queries, 0 error handling) and `invite-friends.tsx` (1 query, 0 error handling) will silently fail on network errors.
