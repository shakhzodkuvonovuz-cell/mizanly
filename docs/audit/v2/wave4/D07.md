### Systemic Patterns

1. **StyleSheet hardcodes dark theme colors everywhere**: All 5 screens define static styles with `colors.dark.*` and `colors.text.*`, then override some (but not all) with `tc.*` in JSX. This is fragile -- any missed override breaks light theme.
2. **Missing haptic feedback**: 3 of 5 screens (chat-export, chat-lock, chat-theme-picker) do not import `useContextualHaptic` at all.
3. **Missing error handlers on mutations/async**: Multiple mutations in chat-folders have no `onError`. Async callbacks in chat-lock and chat-theme-picker lack try/catch.
4. **No StatusBar configuration**: None of the 5 screens set StatusBar style.
5. **Font system inconsistency**: Several screens use `fontWeight: '500'/'600'/'700'` raw instead of `fontFamily: fonts.*`.
