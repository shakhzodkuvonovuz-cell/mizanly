### Key Patterns Across All 5 Screens

1. **Light theme completely broken (all 5 screens):** Every screen uses `colors.text.primary`/`secondary`/`tertiary` (dark-mode-only constants) instead of `tc.text.*` (theme-aware). Light theme users see white text on white backgrounds.

2. **RTL not implemented (3 of 5 screens):** `quran-reading-plan.tsx`, `quran-room.tsx`, and `quran-share.tsx` do not destructure `isRTL` and use no RTL layout utilities. Only `quiet-mode.tsx` and `qr-scanner.tsx` (via `useTranslation`) handle RTL.

3. **Mutations fail silently (2 screens):** `quiet-mode.tsx` and `quran-reading-plan.tsx` have mutations with no `onError` callbacks and no toast feedback.

4. **SafeArea missing (2 screens):** `quran-room.tsx` has no SafeAreaView at all. `quran-share.tsx` imports it but never uses it.

5. **No haptics (2 screens):** `qr-scanner.tsx` and `quran-share.tsx` don't import or use `useContextualHaptic` at all.
