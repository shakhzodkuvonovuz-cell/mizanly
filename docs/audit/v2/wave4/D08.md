### Critical Patterns Across All 5 Screens

1. **`useContextualHaptic` missing from 4 of 5 screens** (circles, close-friends, collab-requests, communities). Only `chat-wallpaper.tsx` uses it. Per project rules, this hook is mandatory.

2. **`fontWeight` raw strings instead of `fontFamily: fonts.*`** in 4 of 5 screens. `circles.tsx`, `close-friends.tsx`, `collab-requests.tsx`, and `communities.tsx` all use fontWeight strings. Only `chat-wallpaper.tsx` uses `fonts.*` properly.

3. **Hardcoded dark-theme colors in StyleSheet never overridden inline** -- particularly tab text, category pills, stats text, and button text. These will break on any future light theme.

4. **`communities.tsx` is the weakest screen**: manual state management instead of react-query, broken pagination (cursor param exists but never passed), silent error swallowing on join/leave, no error UI, `setTimeout` for fake animation delays, and no haptic feedback anywhere.

5. **`collab-requests.tsx` has Pressable-inside-LinearGradient anti-pattern** on accept/decline buttons, reducing tap targets.
