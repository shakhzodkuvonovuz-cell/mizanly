# B09 — Channel, ChannelPost, ChannelSubscription, Community, CommunityNote

**Scope note:** `ChannelSubscription` and `Community` Prisma models do not exist. The actual models are `Subscription` (channel subscriptions), `ChannelMember` (broadcast channel), `CommunityNote`, `CommunityNoteRating`, `CommunityRole`, `CommunityTreasury`, plus many community-related models (LocalBoard, Mentorship, StudyCircle, FatwaQuestion, VolunteerOpportunity, IslamicEvent, UserReputation, VoicePost, WatchParty, SharedCollection, WaqfFund). Audit covers all related service files.

| 3 | H | community-notes.service.ts:57-81 | `rateNote` performs 3 separate DB operations without `$transaction`: (1) create rating, (2) increment vote count, (3) update status.
